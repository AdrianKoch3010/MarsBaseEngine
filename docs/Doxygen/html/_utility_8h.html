◆ MBE_NAME_OF

#define MBE_NAME_OF (   x)    #x

Returns x as a c-string or rather embeds x in quotation marks

This is useful when a unique typename is required. One option to make sure that the name is unique is to use the name of the class. This functions capsules this. The use of MBE_NAME_OF() should always be preferred over typing the classname as a string since the IDE can help prevent spelling mistakes and provide useful functionality such as renaming the class whose name is needed in string form.

Attention
When using MBE_NAME_OF() for the above described scenario, make sure that in order to ensure uniqueness always the full type name is used. This includes e.g. namespaces.
