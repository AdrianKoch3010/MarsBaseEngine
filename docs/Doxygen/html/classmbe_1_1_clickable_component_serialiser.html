Detailed Description

Component serialiser for the mbe::ClicableComponent.

The default values are the same as in the mbe::ClickableComponent

XML format

<Component type="ClickableComponent>
<!-- Optional - default: true -->
<AbsorbeClick>bool</AbsorbeClick>
<!-- Optional - default: true -->
<BubbleUp>bool</BubbleUp>
</Component>
