bool mbe::TextureWrapper::loadFromFile ( std::string  filePath,
bool  createPixelMask = false 
)

Loads a texture from a file and optionally creates a pixel mask for it.

The creating of a pixel mask can be fairly computationally expensive. Hence, this functions allows for calculating it at the same time as loading the textures (in the program flow, not actually in parallel).

Parameters
filePathThe file path of the texture to be loaded
createPixelMaskIf true, a pixel mask is created after loading the texture. If false, no further actions are taken
Note
This method has the same signature as the methods provided by SFML for loading assets from a file such as sf::Texture::loadFromFile(), sf::SoundBuffer::loadFromFile(). This enables the texture wrapper to be treated just like any other asset. Therefore, texture wrappers can be stored in an mbe::TextureWrapperHolder
