Detailed Description

Component serialiser for the mbe::TransformComponent.

XML format

<Component type="TransformComponent">
<Position x="float" y="float" />
<Origin x="float y="float" />
<Scale x="float" y="float" />
<Rotation>float</Rotation>
</Component>
