Detailed Description

Stores a transformation matrix and provides a convenient interaface.

It is esentially a wrapper around the sf::Transformable class. The underlying transformable can be retrieved using the GetTransformable() methdod.
The transform is relative to the parent entity. To get the actual position and transform, use the GetWorldPosition() and GetWorldTransform() methods.

void mbe::TransformComponent::Move ( const sf::Vector2f &  offset)

Move the entity's transform component by a given offset.

This method adds to the current position of the object, unlike SetPosition which overwrites it. Thus, it is equivalent to the following code:

auto position = entity.GetComponent<mbe::TransformComponent>().GetPosition();

void mbe::TransformComponent::SetOrigin ( const sf::Vector2f  origin)
inline

set the local origin of the object

The origin of an object defines the center point for all transformations (position, scale, rotation). The coordinates of this point must be relative to the top-left corner of the object, and ignore all transformations (position, scale, rotation). The default origin of a transformable object is { 0.f , 0.f }.
