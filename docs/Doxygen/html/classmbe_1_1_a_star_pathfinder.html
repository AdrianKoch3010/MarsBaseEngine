Detailed Description

template<class TTileMap>
class mbe::AStarPathfinder< TTileMap >

Class that can be used to perform a star pathfinding on a tile map.

Template Parameters
TTileMapThe type of tile map the pathfinding is performed on. It must provide certain functions with the following signatures:
- Position GetSize() const
- float GetTileMovementSpeed(unsigned int x, unsigned int y)
- std::vector<Position> GetReachableTiles(unsigned int x, unsigned int y)
- typedef Position with an x and a y member.
