Register a serialiser for an object.

The serialiser is added for an object of a particular type. This serialiser will then be used to load / add and store an object of that type from an to an XML element.

Template Parameters
TSerialiserThe type of the serialiser to create and add
TObjectThe type of the object for which the serialiser's store and load function will be applied
TArgumentsThe type of the arguments passed
Parameters
objectNameA string id for the type of the object. This will be the same as the type attribute in the object's xml node
argumentsThe arguments passed to the constructor of the TSerialiser
Note
The serialiser must inherit from TBaseSerialiser
