Detailed Description

Changes a texture wrapper component's subrect over time.

This class stores multiple frames that represent the sub-rectangle of a texture. The resulting animation consists of a sequence of frames that are drawn one after another.

Add several frames of equal size and default origin.

Assumes that all frames are equal in size and are located next to each other (left to right) on the sprite sheet texture. The relative duration will be set to 1 meaning that all frames will have the same duration.

Parameters
sizeThe size of each sub-rect
firstThe position (top-left) of the first sub-rect
numberOfFramesThe number of frames to be added
Warning
Make sure that the sprite sheet texture (to which the animation is later applied) is big enough!
