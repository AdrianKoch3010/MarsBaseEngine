mbe::event::EntityClickedEvent::EntityClickedEvent ( )

Default constructor.

When using this constructor the entity id is set to null making sure that no entity exists under that id. The mouse button is set to sf::Mouse::Button::ButtonCount.

Note
When using the default constructor, the entity id and mouse button should be set before raising the event. This can be done using the mbe::EntityClickedEvent::SetEntityID() and mbe::EntityClickedEvent::SetMouseButton() methods respectively.
