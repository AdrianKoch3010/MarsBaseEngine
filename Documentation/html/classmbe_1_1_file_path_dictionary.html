Detailed Description

template<typename TID = std::string>
class mbe::FilePathDictionary< TID >

Wrapper around the std::unordered_map<TID, std::string> class.

Defines methods for inserting and accessing file paths. There is a template overload for TID == std::string that normalises the id string

Template Parameters
TIDTd type that must be usable as an id in a std::unordered_map
