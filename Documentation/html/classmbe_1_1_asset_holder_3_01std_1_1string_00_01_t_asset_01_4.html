Detailed Description

template<class TAsset>
class mbe::AssetHolder< std::string, TAsset >

A class managing the loading and access of assets.

Template overload of the mbe::AssetHolder class when using a std::string as an id

Template Parameters
AssetThe type of asset to be managed (e.g. sf::Texture). The Asset must provide a loadFromFile() function
Attention
The id strings are not case sensitive! This is to reduce the likelyhood of mistyping an id which may cause unwanted behaviour.

Loads an asset from a file and adds it to the asset dictionary.

Every asset requires a file path. Any additional arguments can be passed using the variadic template.

Template Parameters
TArgumentsThe types of the additional parameters passed to the asset's loadFromFile() method
Parameters
idThe id that can later be used to access this asset. It is not case sensetive so capital letters do not matter. Only use ASCII strings!
filePathThe file path from which the asset will be loaded
argumentsThe additional Parameters for the asset's loadFromFile() function (e.g. fragment shader for sf::Shader)
Exceptions
