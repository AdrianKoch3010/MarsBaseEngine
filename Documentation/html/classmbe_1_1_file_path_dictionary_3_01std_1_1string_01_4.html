Detailed Description

template<>
class mbe::FilePathDictionary< std::string >

Wrapper around the std::unordered_map<std::string, std::string> class.

Defines methods for inserting and accessing file paths. This is a template overload of the mbe::FilePathDictionary<TID> class where TID == std::string.

Template Parameters
TIDTd type that must be usable as an id in a std::unordered_map
Attention
The id strings are not case sensitive! This is to reduce the likelyhood of mistyping an id which may cause unwanted behaviour. Furthermore only ASCII characters should be used.
