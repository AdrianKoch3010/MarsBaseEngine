Detailed Description

template<typename TData>
class mbe::detail::Singleton< TData >

A template class for a singleton.

The class that inherits from singleton must add this as a friend (for the instance function to access the protected / private constructor).
The Singleton is created the time the Instance() method is called.

Template Parameters
TThe type of the singleton object instance
